# 경영캡스톤 중간발표 코드 설명

### 1. 주제
- 이 코드의 주된 목적은 **주가 데이터를 다운로드하고 파일로 저장**하여 이후의 분석에 사용할 수 있도록 준비하는 것이며, 경영캡스톤 중간발표를 위한 초기 데이터 수집 및 처리 작업을 수행함.

---

### 2. 사용된 라이브러리
- **yfinance**: Yahoo Finance에서 주가 데이터를 다운로드하는 데 사용.
- **pandas**: 데이터 프레임을 사용하여 데이터를 다루기 위한 라이브러리.
- **requests**: 웹 요청을 보내기 위한 라이브러리.
- **beautifulsoup4**: 웹 페이지에서 데이터를 스크래핑하기 위한 라이브러리 (추후 활용 가능성).
- **html5lib**: HTML을 파싱하기 위한 라이브러리.
- **lxml**: XML 및 HTML 문서를 처리하는 라이브러리.

---

### 3. 코드 흐름

#### 3.1 패키지 설치 및 불러오기
- `pip`을 사용해 필요한 패키지를 설치하고, 이를 활용해 데이터를 처리할 준비를 함.
- 설치된 주요 패키지:
  - `yfinance`, `numpy`, `pandas`, `lxml`, `beautifulsoup4`, `html5lib`, `requests`, `pytz`

#### 3.2 주가 데이터 다운로드
- **yfinance 라이브러리**를 사용하여 특정 주식의 주가 데이터를 다운로드.
- 주가 데이터는 **날짜별로 기록**되며, 날짜별 **주가 정보(시가, 종가, 고가, 저가, 거래량 등)**가 포함됨.
- 다운로드한 데이터는 `pandas` 데이터프레임으로 변환되어 이후 분석과 시각화에 사용될 수 있음.

#### 3.3 데이터 저장
- 다운로드한 주가 데이터를 **CSV 파일**로 저장하는 코드 포함.
  - CSV 파일 형식은 데이터를 이후 다양한 환경에서 다시 불러와 분석할 수 있도록 호환성을 높임.
  
#### 3.4 데이터 전처리 (추가 가능성)
- 현재 단계에서는 주가 데이터를 수집하고 저장하는 작업이 주로 이루어졌지만, 이후 전처리 작업이 필요할 수 있음.
  - 결측치 처리, 이상치 제거 등의 전처리가 포함될 가능성 있음.

---

### 4. 향후 작업 예상
- **데이터 탐색적 분석(EDA)**: 다운로드한 주가 데이터에 대한 탐색적 분석을 수행하여 기본적인 통계 정보를 확인하고, 시각화를 통해 주가 변화 패턴을 파악할 수 있음.
- **주가 예측 모델링**: 이후 단계에서는 머신러닝이나 딥러닝 모델을 적용하여 주가 예측을 수행할 수 있음. LSTM, Prophet 등 시계열 모델이 유력하게 사용될 가능성 있음.
- **비즈니스 인사이트 도출**: 주가 데이터를 기반으로 비즈니스 관련 의사결정에 도움이 되는 인사이트를 도출할 수 있음.

---

### 5. 결론
이 코드 파일은 경영캡스톤 프로젝트 중간발표를 위한 데이터 수집 및 저장 과정의 일부로, 주가 데이터를 자동으로 수집하고 파일로 저장함으로써 이후의 분석 및 모델링을 준비하는 중요한 작업임.


---


## 2. 파일 설명

### 1. **Clustering.ipynb**
- **주요 내용**: 클러스터링 알고리즘을 사용하여 주식 데이터를 그룹화.
  - **K-평균 클러스터링**: 주식 데이터를 `PER`, `Beta`, `RSI`와 같은 금융 지표를 기준으로 클러스터링하고, 각 클러스터의 중앙값을 분석.
  - **결과 분석**: 각 클러스터의 특성을 분석하여, 주식 간의 유사성을 시각적으로 확인.

### 2. **data download.ipynb**
- **주요 내용**: 주식 데이터를 다운로드하고 로컬에 저장.
  - **yfinance 라이브러리**를 사용하여 주식 데이터를 다운로드하고, CSV 파일로 저장.
  - S&P 500 상위 주식의 데이터를 가져와 분석 준비.

### 3. **data preprocessed.ipynb**
- **주요 내용**: 다운로드된 데이터를 전처리하는 과정을 다룸.
  - 결측치 처리 및 이상치 제거.
  - 다양한 주식 티커의 금융 데이터를 하나의 데이터프레임으로 통합.

### 4. **경영캡스톤 중간발표 코드1.ipynb**
- **주요 내용**: 중간발표를 위한 주식 데이터 처리 및 저장.
  - `yfinance`를 사용하여 주요 주식 티커에 대한 데이터를 다운로드하고, 필요한 지표를 CSV로 저장.

### 5. **Clustering.py**
- **주요 내용**: 클러스터링 알고리즘을 Python 스크립트로 구현.
  - **계층적 클러스터링**을 사용하여 주식 데이터를 그룹화하고, 덴드로그램을 통해 결과를 시각화.
  - `ward` 연결 방식을 사용해 주식 간의 거리를 계산하고, 클러스터링 결과를 시각적으로 분석.

### 6. **data download.py**
- **주요 내용**: 주식 데이터를 다운로드하는 Python 스크립트.
  - S&P 500 상위 100개의 주식 데이터를 다운로드하고, 이를 CSV 파일로 저장.
  - `yfinance` 라이브러리를 사용하여 주가 데이터를 다운로드하고 저장하는 기능을 구현.

### 7. **app.py**
- **주요 내용**: Flask를 사용해 웹 애플리케이션을 구현.
  - **API 경로**: `/api/optimize` 경로를 통해 클러스터링 데이터를 받아 처리하고, 결과를 반환.
  - **Flask 서버**: 서버는 로컬에서 실행되며 클러스터링 결과를 웹 상에서 확인할 수 있도록 제공.

### 8. **index.html**, **styles.css**, **scripts.js**
- **주요 내용**: 웹 애플리케이션의 프론트엔드 구성 파일.
  - **index.html**: 웹 페이지의 기본 구조를 정의하며, 사용자가 데이터를 입력하고 결과를 확인할 수 있는 페이지.
  - **styles.css**: 웹 페이지의 스타일을 정의하는 CSS 파일.
  - **scripts.js**: 웹 페이지에서 동작하는 자바스크립트 코드, API 요청을 처리하고 결과를 화면에 표시.
